Common CSS Properties And Table references

Text Styling

Property Value Description
text-align left, right, center, justify The text-align property is used to control how text is aligned within its containing content box.
text-transform capitalize The text-transform CSS property specifies how to capitalize an element's text.
letter-spacing normal, length The letter-spacing CSS property sets the horizontal spacing behavior between text characters..

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<tr> Table Row The table row container.
<th> Table Head Defines a cell as header of a group of table cells.
<td> Table Data The table data element defines a cell of a table that contains data. .
